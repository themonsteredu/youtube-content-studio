# Testing

기본 검증 명령은 `npm run lint`, `npm run typecheck`, `npm run build`다.

수동 시나리오:

1. 데모 모드에서 세 관리자 화면이 모두 열린다.
2. 제목 변경이 썸네일과 인트로에 즉시 반영된다.
3. 이미지 선택과 강조 색상 변경이 두 미리보기에 반영된다.
4. PNG 버튼이 1920×1080 파일을 내려준다.
5. MP4 버튼이 작업 ID를 반환하고 이력에 기록한다.
6. Supabase 운영 모드에서 비로그인, 일반 사용자, 관리자 권한이 각각 차단·허용된다.
7. 모바일과 데스크톱에서 메뉴, 폼, 미리보기가 겹치지 않는다.
